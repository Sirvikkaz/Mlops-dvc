1. create a git repo and clone it to local
2. create a mycode.py file and add some codes to it 
3. do git add commit, before initializing dvc
# 4. pip install dvc
5 we do "dvc init" . (this creates .dvc, .dvcignore)
6. now do S3 mkdir (creates new S3 directory)
7. Now we do "dvc remote add -d myremote S3" 
8. next is "dvc add data/" and them we add git .ignore and data.dvc
9. now "dvc commit" and then "dvc push"
10. Do git add-commit-push to mark this stage as the first version of the data
11. make changes in the mycode.py, append new row to the data and check changes via dvc status
12. again run dvc commit and then dvc push
13. Then git add-commit-push (this will be version 2)
14. do dvc/git status
15. step 11-14 is how changes is made versions are updated
